How to Use IP Filter Exchange

Quick Start

  1. Paste your proxy in the text box
  2. Click "Parse & Apply" to fill fields automatically
  3. Click "Connect" button
  4. Wait 3-5 seconds for connection to establish

Proxy Format

Format: hostname:port: username:password

Example:

proxy.example.com:8080:user123:pass456

Manual Entry

If you prefer, you can enter proxy details manually in the four fields below the text box:

  • Hostname: The proxy server address
  • Port: The proxy server port number
  • Username: Your proxy username
  • Password: Your proxy password

Connection Status

When successfully connected, you will see:

  • Your proxy IP address
  • Detected timezone
  • Location information (if available)
  • Button changes to "Disconnect"

Proxy Type (Settings)

Choose the appropriate proxy type for your connection:

  • HTTP: Most common, works with most proxies (Default)
  • HTTPS: Encrypted HTTP proxy connection
  • SOCKS5: Advanced protocol with better performance

Timezone Setting

Control how your browser timezone is handled:

  • Based on Proxy (Recommended): Automatically matches browser timezone to proxy location for better privacy
  • Real Timezone: Uses your actual system timezone

WebRTC Protection

Protect against IP leaks:

  • Disabled (Recommended): Blocks WebRTC to prevent your real IP from leaking
  • Enabled: Allows WebRTC (may leak your real IP through browser features)

Switching Proxies

To switch to a different proxy:

  1. Click "Disconnect" if currently connected
  2. Paste or enter the new proxy details
  3. Click "Connect"
  4. The old connection will be automatically cleared

Clear Cache & Data

If your proxy keeps repeating or not connecting to new proxies properly:

  1. Go to Settings (gear icon)
  2. Scroll to "Clear Cache & Data" section
  3. Click "Clear All Cache & Data" button
  4. This removes all cached connections, cookies, local storage, and service workers
  5. Allows you to connect to different proxies without interference

Troubleshooting

  • Connection Failed: Verify proxy credentials and try again
  • Repeated Proxy: Use "Clear All Cache" in Settings
  • Slow Connection: Try a different proxy type or server
  • IP Leak: Ensure WebRTC Protection is set to "Disabled"
